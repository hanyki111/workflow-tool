# AI 워크플로우 엔진 소개

AI 워크플로우 엔진 튜토리얼에 오신 것을 환영합니다!

## Workflow-Tool이란?

AI 워크플로우 엔진은 AI 지원 개발을 위해 설계된 구조화된 워크플로우 관리 시스템입니다. 개발 과정 전반에 걸쳐 규율과 일관성을 유지하는 데 도움을 줍니다.

## 핵심 개념

### 1. 스테이지
워크플로우는 스테이지로 나뉩니다:
- **마일스톤 스테이지 (M0-M4)**: 고수준 프로젝트 단계
- **페이즈 스테이지 (P1-P7)**: 상세 구현 단계

### 2. 체크리스트
각 스테이지에는 다음 스테이지로 이동하기 전에 완료해야 하는 항목들의 체크리스트가 있습니다.

### 3. 전이
스테이지 간 이동은 모든 체크리스트 항목 완료와 추가 조건 충족을 필요로 합니다.

## 왜 이 도구를 사용하나요?

1. **프로세스 규율**: 중요한 단계를 건너뛰지 않도록 보장
2. **컨텍스트 인식**: AI 어시스턴트가 워크플로우 상태를 읽을 수 있음
3. **문서화**: 모든 전이가 기록됨
4. **품질 관리**: 내장된 리뷰 스테이지

## 빠른 시작

```bash
# 현재 상태 확인
flow status

# 항목 완료 표시
flow check 1 2 3

# 다음 스테이지로 이동
flow next
```
